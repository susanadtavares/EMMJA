# EMMJA - Escola de Música Management System

Aplicação PERN (PostgreSQL, Express.js, React, Node.js) para gestão de escola de música.

## 📋 Perfis de Utilizador
- **Admin**: Gestão completa do sistema
- **Docente**: Gestão de alunos e aulas
- **Aluno/Encarregado**: Consulta de informações pessoais

## 🗂️ Estrutura do Projeto
- `backend/` - Servidor Node.js + Express
- `frontend/` - Aplicação React

## 🚀 Como Começar

### Backend
```bash
cd backend
npm install
npm run migrate
npm run start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## 🔐 Tecnologias
- **Database**: PostgreSQL + Sequelize ORM
- **Backend**: Node.js, Express.js, JWT
- **Frontend**: React, Tailwind CSS
